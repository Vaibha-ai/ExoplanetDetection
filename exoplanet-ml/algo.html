Exoplanet Detection Algorithms

Algorithm 1: Astro CNN

A Convolutional Neural Network of the following architecture can detect presence of an exoplanet by first Phase Folding Light Curves and then looking for any dips in them.

Algorithm 2: Exoplanet Detection using Recurrence Measures

A Recurrence plot is a plot showing, for each moment in time, the times at which the state of a dynamical system returns to the previous state at. By feeding the Recurrence plot and its measures to CNN Models it is possible to detect presence of Exoplanets without the use of any Phase Folding.
